快速排序

Nov 18, 2018 | 算法

挖坑填数+分治法

  1. 在数据集之中，选择一个元素作为“基准”
  2. 按照顺序，与基准比较，所有小于基准元素放在左侧，所有大于基准元素都放在右侧
  3. 对“基准”左边和右边的两个子集，不断重复第一步和第二步，直到所有子集只剩下一个元素为止。

性能

不稳定

时间复杂度：平均n*log(n)， 最坏：另外一边总是为空 O(n^2) 最好： 每次分区都很均匀 O(nlogn)

空间复杂度： 递归树深度 最好O(log2n) 最差O(n) 平均O(logn) ### 实现

function quickSort(arr){    
    if(arr.length <= 1){
        return arr;
    }
    let pivotIndex = parseInt(arr.length/2);
    // let pivot = arr[pivotIndex]; 这样会造成call stack溢出
　　var pivot = arr.splice(pivotIndex, 1)[0];
    let right = [];
    let left = [];
    for(let i=0;i<arr.length;i++){
        if(arr[i] < pivot){
            left.push(arr[i]);
        }else{
            right.push(arr[i]);
        }
        console.log(left.concat([pivot],right));
    }
    return quickSort(left).concat([pivot], quickSort(right));
}
